Hestia

Ques es

Hestia es una ampliación diseña para poder ayudar a los usuarios a organizar y clasificar sus archivos de una forma rápida y concisa.

Alguna opciones principales son:

  • Compresor y Descompresor
  • Convertidores
  • Descargar videos de youtube

Uso

Si es la primera vez que abres la aplicación, Hestia descargar algunos archivos para su buen funcionamiento, la aplicación lo mandara a una pagina web donde explica los pasos a paso para poder hacer configuración correctamente. Si no es la primera vez que abres la aplicación y tiene algunas dudas o fallos puede dirigirse a la siguente pagina hay encontrar el manual de la aplicación o puedes copiarnos un comentario si tienes algún fallo para poder solucionar.
